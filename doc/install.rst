How to install?
===============

Requirements
------------

* python3
* rsync

Package manager
---------------

An AUR package `AUR package <https://aur.archlinux.org/packages/python-vitalus/>`_ is available. 

PyPI
----

`See Pypi <http://pypi.python.org/pypi/Vitalus/>`_

To install with pip:

.. code-block:: sh

    pip install Vitalus


Manual installation
-------------------

.. code-block:: sh

    python setup.py --root=/usr/local/bin
